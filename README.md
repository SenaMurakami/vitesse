# Vitesse

This theme is a WordPress theme with Vite for fast development.

## Installation

```terminal: terminal
git clone https://github.com/SenaMurakami/vitesse.git
cd vitesse
npm install
```

## Usage

1. Start a project with Docker, Mamp, etc.
2. `npm run dev`
3. View [http://localhost:3000/](http://localhost:3000/)
